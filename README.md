# Student Helper Python Project

A simple Python desktop/terminal app that helps students manage study tasks, calculate GPA, create quizzes, track notes, and plan study sessions.

## Features

- Student profile setup
- Task manager with priority and due dates
- Notes manager
- GPA/marks calculator
- Quiz practice system
- Study timetable planner
- Progress dashboard
- Data saved locally in JSON files

## Requirements

- Python 3.9 or higher
- No external libraries required

## How to Run

```bash
python main.py
```

## Files

- `main.py` - main program
- `storage.py` - saves and loads data
- `tasks.py` - task manager
- `notes.py` - notes manager
- `gpa.py` - GPA/marks calculator
- `quiz.py` - quiz system
- `planner.py` - study planner
- `dashboard.py` - progress dashboard
- `data/` - stores saved data

## Default Quiz Topics

The app includes basic Python and study skills quiz questions. You can add more inside `quiz.py`.
